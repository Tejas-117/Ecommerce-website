## Overview

### Project

Ecommerce Website for computer parts and other electronics

- Users (Customers) can 
  - view products.
  - make orders and pay for them.
  - add and delete their reviews. 

- Admins can 
  - add/edit/delete products.
  - delete/fulfill orders.
  - delete users.
